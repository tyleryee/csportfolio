Specific Tags and Functions

In order to write a paragraph, you have tags which allows you to write and appear on the page. The paragraph tags allows you to type a paragraph or text into your webpage. In order to have text on your webpage you need to type in-between the tags to create an element which will display on the webpage.

To have headers on your page, you need to know that their are six sizes (h1-h6). H1 should be the main header since it is the biggest out of all of the headers while h6 is the smallest header. To create a header you need to type in-between the tags to make an element.
